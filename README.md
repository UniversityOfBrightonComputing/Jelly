Jelly
=====

Jelly is a simple(-ish) Mario style game which can be used
to learn intermediate/advanced game programming techniques

Compile & Run
=============

Required:
- JDK 1.8+
- almasb-lib.jar 1.1+ (can be found in lib/)

Commands:

```bash
$ mvn package
```

WINDOWS:
```bash
$ java -cp "lib/*;target/*" com.almasb.jelly.v2.Main
```

MAC OS/LINUX:
```bash
$ java -cp "lib/*:target/*" com.almasb.jelly.v2.Main
```
